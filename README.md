## 🧭 مأموریت

«اکونوژین» سامانه‌ای کاملاً متن‌باز، رایگان و مقاوم در برابر تحریم است که داده‌های میدانی، ماهواره‌ای و هواشناسی را به ابزارهای تصمیم‌گیری برای جوامع محلی، تسهیلگران و مدیران پروژه‌های **مدیریت جامع آبخیز** تبدیل می‌کند. این پلتفرم نقش **هاب مرکزی** را برای **هیدرومانوژین** ایفا می‌کند و کلیهٔ فرایندهای پایش (MRV)، آموزش (FFS)، مدیریت دانش (KMS) و گزارش‌دهی شفاف به سرمایه‌گذاران بین‌المللی (GCF) را پشتیبانی می‌نماید.

---

## 🏗️ معماری کلی (High-Level Architecture)

سامانه بر اساس معماری **رویدادمحور (Event-Driven)** و **میکروسرویس‌های مستقل** طراحی شده است.

| لایه | فناوری‌ها |
| :--- | :--- |
| **جمع‌آوری داده** | Google Earth Engine (ماهواره‌ای)، KoboToolbox (میدانی آفلاین)، Open-Meteo (هواشناسی)، MQTT (سنسورهای زمینی) |
| **پردازش و پشتیبان** | FastAPI (REST API)، Celery + Redis (وظایف زمان‌بندی‌شده و تحلیل سنگین) |
| **ذخیره‌سازی** | PostgreSQL 17 + PostGIS 3.5 (داده‌های مکانی و ساختاریافته)، Redis (کش)، MinIO (فایل‌های حجیم) |
| **پیشخوان** | React 19 + TypeScript، Zustand مدیریت حالت، React-Leaflet نقشه، Recharts نمودار، Tailwind CSS |
| **ارتباطات** | Nginx (API Gateway)، JWT (احراز هویت)، Webhook (KoboToolbox) |
| **استقرار** | Docker Compose (توسعه)، Podman (تولید بدون تحریم)، GitHub Actions (CI/CD) |

## ✨ ویژگی‌های کلیدی

- **🛰️ پایش هوشمند (MRV):** محاسبهٔ خودکار شاخص‌های NDVI، EVI، SPI، VHI از تصاویر Sentinel-2 و MODIS، به‌روزرسانی هر ۵ روز با Celery Beat.
- **📱 جمع‌آوری میدانی آفلاین:** یکپارچه با KoboToolbox برای ثبت مشاهدات کشاورزان بدون نیاز به اینترنت، همگام‌سازی خودکار.
- **🎓 مدارس مزرعه‌ای (FFS):** مدیریت دوره‌ها، جلسات هفتگی، ردیابی حضور، مزارع یادگیری و ثبت نوآوری‌های بومی با گردش کار تأیید.
- **📚 مدیریت دانش (KMS):** کتابخانهٔ پروتکل‌های استاندارد (SOP)، دائرةالمعارف گونه‌های بومی، درس‌آموخته‌ها و تالار گفتگوی محلی.
- **⚠️ هشدار زودهنگام (FbF):** تحلیل خشکسالی با SPI-3 و ارسال هشدار پیامکی به کشاورزان ناظر و کارگروه‌های منظر.
- **🌊 مدل‌سازی هیدرولوژیک:** اولویت‌بندی زیرحوزه‌ها با مدل M-PRI، شبیه‌سازی رواناب و فرسایش (RUSLE)، بهینه‌سازی الگوی کشت با AquaCrop.
- **🔗 شفافیت مالی (Blockchain):** ثبت هش تغییرناپذیر گزارش‌های MRV روی بلاک‌چین سازگار با EVM برای ممیزی سرمایه‌گذاران بین‌المللی.
- **🧩 هاب سرویس‌ها:** پنل مدیریت مرکزی امکان اتصال ده‌ها ابزار متن‌باز (فروشگاه، کتابخانه، چت، LMS و...) را از طریق iframe و API فراهم می‌کند.

## 🚀 شروع سریع (توسعه‌دهندگان)

### پیش‌نیازها
- Docker & Docker Compose (یا Podman)
- Python 3.12+
- Node.js 20+

### راه‌اندازی با Docker Compose

```bash
# کلون مخزن
git clone https://github.com/narvan2050/eco-nojin-narvan.git
cd eco-nojin-narvan

# تنظیم متغیرهای محیطی
cp .env.example .env
# فایل .env را با کلیدها و رمزهای خود ویرایش کنید

# راه‌اندازی کامل پشته (پایگاه داده، پشتیبان، پیشخوان، MinIO)
docker compose up -d
```

پس از اجرا:
- **پیشخوان (Frontend):** `http://localhost:3000`
- **پشتیبان (Backend API):** `http://localhost:8000/docs` (Swagger)
- **MinIO کنسول:** `http://localhost:9001`
